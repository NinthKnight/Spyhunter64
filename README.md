# Spyhunter64
Antispy不支持64位版本，因此基于Antispy的基础上，加入了64位系统的支持。

1. SpyHunter64是一款方便的反病毒和rootkit工具。它可以检测、分析和恢复各种内核修改，就像ssdt钩子一样。在它的帮助下，您可以轻松地发现和删除系统中的恶意软件。

2. 支持系统: Win7, Win10 1809，1903，19041-19045, 22000, 22621 (64位)

3. 更多功能，后面持续更新中...
